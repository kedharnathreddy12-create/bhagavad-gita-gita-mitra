# Krishna Bhagavad Gita - Simple Telugu Platform

A modern, responsive, and SEO-optimized web application built to bring the wisdom of the Bhagavad Gita to present-day Telugu speakers in simple, easy-to-understand language.

## 🚀 Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Deployment**: Vercel Ready

## ✨ Features
- **Modern UI**: Premium spiritual design with dark blue, gold, and saffron color palette.
- **Glassmorphism**: Beautiful translucent cards and navigation.
- **18 Chapters Structure**: Dynamic routing for all chapters and slokas.
- **Simple Telugu Translation**: Complex Sanskrit and pure Telugu words replaced with everyday usage.
- **Gita Mitra AI**: A built-in floating chatbot to answer doubts based on Gita's teachings.
- **Daily Wisdom**: Dedicated section for daily insights.

## 🛠️ Getting Started

### Installation
Clone the repository and install dependencies:
```bash
npm install
```

### Running Locally
Start the development server:
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Build & Deploy
This project is Vercel-ready. To build for production:
```bash
npm run build
```
To start the production server:
```bash
npm start
```

You can deploy this instantly using [Vercel](https://vercel.com/new).

## 📁 Project Structure
- `src/app/`: Next.js App Router pages (Home, Chapters, Daily Wisdom, About)
- `src/components/`: Reusable UI components (Navbar, Footer, ChapterCard, Chatbot)
- `src/data/`: Mock data containing Chapters, Slokas, and Chatbot responses
- `src/types/`: TypeScript definitions
